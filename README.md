# 📊 E-Commerce Sales Data Analytics Dashboard

## Project Overview

This project is an interactive **E-Commerce Sales Data Analytics Dashboard** developed using **Python** and **Streamlit**. It analyzes sales data to generate meaningful business insights through data cleaning, visualization, and interactive dashboards. The project helps identify sales trends, customer behavior, top-selling products, and payment method preferences. It also exports cleaned datasets and analysis reports for future use.

## 🌐 Live Dashboard

**Live Demo:** https://retail-sale-dashboard.streamlit.app/

## Features

* Data Cleaning and Preprocessing
* Missing Value & Duplicate Analysis
* Exploratory Data Analysis (EDA)
* Sales, Customer, and Product Analysis
* Payment Method Analysis
* Interactive Dashboard using Streamlit
* Charts and Visualizations using Plotly & Matplotlib
* Export Cleaned Dataset to Excel
* Business Insights Generation

## Technologies Used

Python, Pandas, NumPy, Matplotlib, Plotly, Streamlit, OpenPyXL, and VS Code.
