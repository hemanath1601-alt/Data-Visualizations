# E-Commerce Sales Data Analytics Project

## Project Overview

This project analyzes an E-Commerce Sales dataset using Python. The project performs data cleaning, exploratory data analysis (EDA), business insights generation, and interactive dashboard visualization.

## Features

* Data Cleaning
* Missing Value Analysis
* Duplicate Removal
* Exploratory Data Analysis (EDA)
* Sales Analysis
* Customer Analysis
* Product Analysis
* Payment Method Analysis
* Interactive Dashboard
* Excel and CSV Export
* Data Visualization

## Technologies Used

* Python 3.10+
* Pandas
* NumPy
* Matplotlib
* Plotly
* Streamlit
* OpenPyXL
* VS Code

## Dataset

The dataset contains information about e-commerce orders, including:

* Order ID
* Date
* Customer ID
* Product
* Quantity
* Unit Price
* Shipping Address
* Payment Method
* Order Status
* Tracking Number
* Items in Cart
* Coupon Code
* Referral Source
* Total Price

## Installation

Install the required Python packages:

```bash
pip install pandas numpy matplotlib plotly streamlit openpyxl
```

Or install from the requirements file:

```bash
pip install -r requirements.txt
```

## How to Run

### Run Data Analysis

```bash
python analysis.py
```

### Run Dashboard

```bash
streamlit run dashboard.py
```
